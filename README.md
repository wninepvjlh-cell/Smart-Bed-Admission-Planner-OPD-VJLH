# ระบบจองเตียงผู้ป่วย (Prototype)

ไฟล์นี้เป็นโปรโตไทป์ของหน้าเว็บ `index.html` และสคริปต์ `app.js` ที่พัฒนาตามโค้ดที่คุณให้ไว้

วิธีรันแบบง่าย (ไม่ต้องติดตั้ง Node.js):

1. เปิด Terminal ไปที่โฟลเดอร์โปรเจกต์:

```bash
cd "/Users/thanyaluckpairachsaranon/Desktop/Smart Bed Planner OPD"
```

2. ใช้ Python เพื่อรัน static server (macOS มักมี Python ติดตั้งไว้):

```bash
# Python 3
python3 -m http.server 8000

# หรือ (ถ้าใช้ python เป็น python3)
python -m http.server 8000
```

3. เปิดเว็บเบราว์เซอร์ไปที่:

```
http://localhost:8000
```

หมายเหตุ:
- `data_sdk.js` และ `element_sdk.js` ที่อ้างอิงใน `index.html` เป็น SDK ของระบบเดิม (ถ้ามี hosting หรือ environment ที่ให้มา) — ถ้าไม่มีไฟล์ SDK เหล่านี้ หน้าเว็บจะยังทำงานเป็น UI แบบหน้ากระดาษ (form/อินเตอร์เฟซ) แต่ฟีเจอร์เชื่อมต่อ backend (create/update/delete) จะเป็น mock และไม่เก็บข้อมูลจริง
- ถ้าต้องการให้บันทึกข้อมูลจริง ให้บอกว่าต้องการเชื่อมกับ API แบบไหน (REST / Firebase / ฐานข้อมูลอื่น) ฉันจะช่วยวางโครงร่างการเชื่อมต่อให้

ต่อไปฉันสามารถ:
- เพิ่ม mock data เพื่อทดสอบ UI
- เชื่อมต่อกับ REST API / Firebase ตามที่คุณต้องการ
- ปรับธีม ฟอนต์ หรือลายเส้นไอคอนให้ดูหรูขึ้นตามที่ร้องขอ

บอกฉันได้เลยว่าต้องการให้ฉันทำขั้นตอนต่อไปไหน (เช่น เพิ่ม mock data และหน้าทดสอบ, เชื่อม API, ปรับสไตล์เพิ่มเติม)
